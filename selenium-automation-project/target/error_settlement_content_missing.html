      Discount Agreements Additional Parameters

      Show Additional Attributes for Agreement set-up

      Additional Attributes allow an end-user to capture agreement parameters with more flexibility. By clicking on Additional Attribute additional columns are shown.

      Balancing displays column 'Balancing'.
      Call Destinations displays columns 'Destination & Call Type' and 'Selected Destinations'.
      Fair Usage displays columns 'Fair Usage Threshold' and 'Fair Usage Rate'.
      Qualifying Services displays columns 'Qualifying Service Type', 'Qualifying Event Type', 'Qualifying Discount Direction', 'Qualification Basis' and 'Qualifying Volume or Percentage'.
      PMN Level Rates displays columns 'Configure PMNs', 'Home PMN(s)' and 'Partner PMN(s)'. This Additional Attribute is only relevant for Group Agreements.
      Airtime & Toll displays columns 'Toll Rate' and 'AirTime Rate'.
      Intermediate Dates displays columns 'Valid From' and 'Valid To'.
      IMSI Caps displays columns 'IMSI Cap Traffic in MB', 'IMSI Cap Value'. By enabling this Additional Attribute, Calculation Types 'Per Day Per IMSI' become available.
      Inbound Market Share displays column 'Inbound Market Share'.
      Segment displays column 'Segment'.

      If Agreement Attribute is unchecked and disabled (greyed out), this means that this functionality is not allowed to the Client due to a limitation in the content of delivered DCH file.
      If Agreement Attribute is checked and disabled (greyed out), this means that some values have been entered in relation to that Attribute and need to be removed before unchecking can be finalised.
